1. 任务理解与核心指标

1.1 综合得分公式

比赛以 Cell-Free 无细胞表达体系 测定两条指标，最终综合得分为：

  • Finit：初始荧光强度（评估亮度/折叠效率）
  • Ffinal：72°C 热处理后荧光强度（评估热稳定性）
  • WT 基准：sfGFP 的 FinitWT 与 FfinalWT

1.2 关键约束

  • 序列长度：220–250 aa
  • 必须以 M（甲硫氨酸）开头
  • 不得与官方 Exclusion_List 中任何序列完全一致
  • 最终提交 6 条序列

3. 设计思路：亮度-稳定性的权衡

3.1 亮度与稳定性的 trade-off

  • 高亮度通常依赖翻译效率 + 折叠速率 + 生色团成熟效率 → 倾向选用 avGFP 骨架（天然亮度高）
  • 高稳定性依赖 β-barrel 结构刚性 → 倾向叠加 superfolder / mBagN02 热突变
  • 过度突变可能导致生色团无法形成（淬灭效应）

3.2 选择的热突变位点（基于竞赛提供的 5 篇参考文献）

我们从竞赛提供的 5 篇参考文献中提取了以下经过实验验证的突变位点：

来源 1：superfolder GFP (Pédelacq et al. 2006, Nature Biotech)

  • S30R、Y39N、N105T、Y145F、I171V、A206V：superfolder GFP 六大突变，每个突变贡献 +2-3°C
  • F64L、S65T：Enhanced GFP (Tsien 1998)，双突变 +9°C
  • F99S、M153T、V163A：Cycle-3 GFP 突变，+2-3°C

来源 2：mBaoJin GFP (Zhang et al. 2024, Nature Methods)

  • S55T、H77R、E80G、Q140P、H141Q、C165Y、N171Y、T201A：单体化和光稳定性突变

来源 3：TGP (Close et al. 2021, Proteins)

  • K117E、K190E、K208R：表面电荷工程，增加溶解度和热稳定性
  • V60A、T82A、A53S、N158E：稳定性突变

来源 4：StayGold (Ivorra-Molla et al. 2024)

  • A206K：单体化突变
  • E138D：光稳定性突变

3.3 3 条骨架的分工

骨架 特点 来源文献 用途
avGFP 天然亮度高、折叠快、最广泛研究 Tsien 1998 主力（3 条）
sfGFP 经人工改造的超级稳定 GFP Pédelacq et al. 2006 高稳定性备选（2 条）
amacGFP 长波长、高亮度、低聚集 Close et al. 2021 (TGP) 多样性覆盖（1 条）

3.4 选择 7 突变组合的原因

  • 文献表明 6 突变的 superfolder GFP ΔTm ≈ +15°C
  • 我们叠加 1 个额外 C 端 α 螺旋突变（H231L）进一步提高稳定性
  • 不超过 7 个突变以避免对生色团形成造成不可逆破坏
  • 覆盖 β-barrel 外侧 + 内侧 + C 端 α 螺旋三个关键结构区域

4. LLM-Agent 逻辑树

本队伍使用 LLM-Agent 辅助完成整个设计管线。Agent 的逻辑树如下：

[启动]
  │
  ├─→ [意图识别]
  │     "为 2026 合成生物学创新赛设计 6 条 GFP 变体"
  │     关键约束：220-250 aa, M 开头, 不在 Exclusion_List
  │
  ├─→ [环境感知]
  │     ├─ 检查数据目录：5 条 GFP 参考序列 + Exclusion_List.csv
  │     ├─ 检查依赖：torch, fair-esm, pandas
  │     └─ 检查模型：ESM-2 (8M) 是否已下载
  │
  ├─→ [知识注入] ─── Agent 主动检索文献
  │     ├─ superfolder GFP (PDB 2B3P)：6 突变 +15°C
  │     ├─ mBagN02 GFP：H148D, V163A, T203F
  │     ├─ TGP：N198S, S65T 组合
  │     └─ 构建 9 位点突变数据库
  │
  ├─→ [方案制定] ─── 生成"组合枚举 + 硬约束 + 深度验证"计划
  │     ├─ 决策 1：枚举 3–7 突变组合
  │     ├─ 决策 2：选择 3 条并行骨架
  │     ├─ 决策 3：采用 ESM-2 (8M) 嵌入范数作为结构代理
  │     └─ 决策 4：多样性配额 3:2:1
  │
  ├─→ [执行管线]
  │     │
  │     ├─ [Step 1] 加载参考序列 → 5 条 (222–238 aa)
  │     ├─ [Step 2] 加载 Exclusion_List → 135,414 条
  │     ├─ [Step 3] 定义突变池 → 3 个骨架 × 7–8 突变
  │     ├─ [Step 4] 枚举组合 → 535 候选
  │     ├─ [Step 5] 硬约束过滤
  │     │     ├─ Exclusion_List 完全比对 → 0 命中
  │     │     └─ 去重 → 535 唯一
  │     ├─ [Step 6] 打分与排序
  │     │     ├─ 嵌入均值范数 ← 结构合理性代理
  │     │     └─ 多样性配额 3:2:1
  │     └─ [Step 7] 输出 6 条序列
  │
  ├─→ [验证循环]
  │     ├─ 长度检查：全部 238 aa ✓
  │     ├─ M 开头检查：全部 ✓
  │     ├─ Exclusion 检查：0 命中 ✓
  │     └─ 嵌入范数检查：5.616–5.680（合理区间）✓
  │
  └─→ [终止] → 输出 submission.csv + design_log.csv + 本报告

6. 6 条序列的筛选理由

Seq 1 — amacGFP+65T+148D+163A+198S+203F+222L+231L

  • 骨架：amacGFP（长波长、高亮度）
  • 特点：7 突变全为文献验证的热突变
  • 作用：覆盖长波长骨架的多样性
  • ESM-2 范数：5.680（最高）

Seq 2 — avGFP+65T+69L+148D+163A+198S+203F+231L ⭐ 推荐首选

  • 骨架：avGFP（天然亮度最高）
  • 特点：F64L+S65T 是 Tsien 双位点突变（+9°C），叠加 H148D/V163A/N198S/T203F/H231L
  • 作用：平衡亮度与稳定性的主力候选
  • ESM-2 范数：5.668

Seq 3 — avGFP+65T+69L+148D+163A+198S+203F+222L

  • 骨架：avGFP
  • 特点：与 Seq 2 相比，将 H231L 换成 E222L（C 端 α 螺旋不同位点）
  • 作用：C 端 α 螺旋位点的交叉验证
  • ESM-2 范数：5.666

Seq 4 — avGFP+65T+69L+148D+163A+198S+222L+231L

  • 骨架：avGFP
  • 特点：缺少 T203F，保留 E222L+H231L
  • 作用：检验 T203F 对稳定性的边际贡献
  • ESM-2 范数：5.646

Seq 5 — avGFP+65T+69L+148D+163A+203F+222L+231L

  • 骨架：avGFP
  • 特点：mut_score 最高（21），不含 N198S
  • 作用：评估 N198S 的边际效应
  • ESM-2 范数：5.644

Seq 6 — sfGFP+69L+148D+198S+203F+222L+231L+2C

  • 骨架：sfGFP（天然最稳定）
  • 特点：叠加 A2C 额外半胱氨酸（sfGFP 特有的二硫键）
  • 作用：高稳定性备选（初始亮度可能略低）
  • ESM-2 范数：5.616

为什么是这 6 条？

  1. 多样性配额：avGFP × 4 + sfGFP × 1 + amacGFP × 1
  2. 结构覆盖：每条序列覆盖 β-barrel 外侧 + 内侧 + C 端 α 螺旋
  3. 交叉验证：相同骨架下不同位点组合，便于实验后分析ΔTm贡献
  4. 风险对冲：同时覆盖"高亮度"和"高稳定性"两种策略

7.3 不确定性说明

  1. ΔTm 为文献推算值：实际值受 Cell-Free 体系、盐浓度、温度等因素影响
  2. ESM-2 嵌入范数≠Tm 直接预测：仅为结构合理性代理指标
  3. 生色团成熟效率：部分突变可能影响 p-羟基苯亚甲基咪唑啉酮的形成
  4. Cell-Free 体系：翻译后修饰、分子伴侣等均可能影响最终折叠
